SnapMind 개인정보처리방침

시행일: 2026년 3월 17일

1. 수집하는 개인정보

SnapMind(이하 "서비스")는 다음 정보를 수집합니다.

  • 계정 정보: Google 로그인 시 이메일 주소, 프로필 이름
  • 사용자 콘텐츠: 업로드한 문서, 입력한 텍스트 (학습 팩 생성 목적)
  • 학습 데이터: 학습 진행률, 퀴즈 응답 기록

2. 개인정보의 이용 목적

  • 학습 팩 생성 및 개인화된 학습 경험 제공
  • 학습 진행 상황 동기화
  • 서비스 개선 및 오류 수정

3. 제3자 제공

서비스는 학습 팩 생성을 위해 사용자가 입력한 텍스트를 OpenAI API로 전송합니다. 전송된 데이터는 AI 모델 학습에 사용되지 않습니다. 그 외 제3자에게 개인정보를 판매하거나 공유하지 않습니다.

  • Supabase: 인증 및 데이터 저장
  • OpenAI: AI 기반 콘텐츠 생성
  • Cloudflare: API 호스팅

4. 데이터 보관 및 삭제

사용자는 언제든지 계정 삭제를 요청할 수 있으며, 요청 시 모든 관련 데이터를 30일 이내에 삭제합니다.

5. 보안

모든 데이터 전송은 HTTPS를 통해 암호화되며, 접근 권한을 최소한으로 제한합니다.

6. 아동 개인정보

본 서비스는 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 변경 사항

본 방침이 변경될 경우 앱 내 공지 또는 이메일로 안내합니다.
